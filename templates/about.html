About This Project

Traffic Sign Recognition using Deep Learning

Project Overview

This is an AI-powered traffic sign recognition system built as a college project in machine learning and computer vision. The system uses a custom Convolutional Neural Network (CNN) trained on a labeled dataset of 53 different traffic sign categories.

The model leverages data augmentation, batch normalization, and dropout regularization to achieve high accuracy and generalization to unseen images.

REST API

You can use the prediction endpoint programmatically:

POST /api/predict
Content-Type: multipart/form-data
Body: file=<image_file>

# Example with curl:
curl -X POST http://localhost:5000/api/predict \
     -F "file=@traffic_sign.jpg"
